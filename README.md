# IntraSwap

Projekt
IntraSwap ist eine Plattform für Schüler, um gebrauchte Schulmaterialien wie Bücher, Taschenrechner oder Kleidung weiterzugeben.

Problem
Viele Materialien werden jedes Jahr neu gekauft, obwohl sie noch brauchbar sind.
Es gibt keine einfache Möglichkeit, diese innerhalb der Schule weiterzugeben.

Lösung
Schüler können Produkte hochladen, ansehen und andere direkt kontaktieren.

Zielgruppe
Schüler an unserer Schule

Features (MVP)
Login mit Schul-E-Mail
Produkt hochladen (Foto + Beschreibung)
Produkte anzeigen
Kontakt aufnehmen

Ziel
Eine einfache Plattform, die Schüler wirklich nutzen und die hilft, Materialien weiterzugeben.
